### Html Lists

---

> **Html Lists allows web developers to group a set of related items in a list**

There are two types of lists -

1. Ordered List - ol
2. UnOrdered List - ul

> _Ordered List_

```html
<ol>
  <li>list item 1</li>
  <li>list item 2</li>
  <li>list item 3</li>
</ol>
```

##### attributes

1. `type = "1", "A", "a", "I", "i"`
2. `start = "20"`

> _unordered list_

```html
<ul>
  <li>list item</li>
  <li>list item</li>
  <li>list item</li>
</ul>
```
